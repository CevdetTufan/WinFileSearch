# Changelog

All notable changes to WinFileSearch will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0-beta.1] - 2024-XX-XX

### Added
- ⚡ **Fast File Search**: Lightning-fast full-text search powered by SQLite FTS5
- 📁 **Folder Indexing**: Add multiple folders to index with drag & drop support
- 🔍 **Search Filters**: Filter by file type (Documents, Images, Media, All)
- 🌙 **Dark Theme**: Modern dark UI with accent colors
- 🌐 **Multi-language Support**: English and Turkish localization
- ⌨️ **Global Hotkey**: Quick access with Win+Shift+F
- 📌 **System Tray**: Minimize to system tray for quick access
- 🔄 **Real-time File Watching**: Automatic index updates when files change
- 📊 **Performance Metrics**: Track search statistics and memory usage
- 📝 **Logging**: Comprehensive logging with Serilog
- 🔄 **Auto-Update**: Check for updates and install directly from the app

### Technical Features
- SQLite FTS5 for fast full-text search
- UI Virtualization for smooth scrolling with large result sets
- Background thread processing to keep UI responsive
- Batch file processing for efficient indexing

## [Unreleased]
